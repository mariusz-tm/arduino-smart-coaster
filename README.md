# Arduino-smart-coaster

Smart coaster powered by an Arduino uno to display active temperature of cup, desired temperature by user, and push notifications when desired temperature is reached



The thermistor B-value and the calculations for the resistance and final temperature value was taken Seeed Studio's "Sketchbook\_Starter\_Kit\_V2.0" repository, who are the creators of the Grove Kit. The repository can be found at:

[https://github.com/Seeed-Studio/Sketchbook\_Starter\_Kit\_V2.0/tree/master](https://github.com/Seeed-Studio/Sketchbook_Starter_Kit_V2.0)



To connect to the internet, an "arduino\_secrets.h" file inside the same folder as the smartcoaster.ino, where #define SECRET\_SSID = "Name of Wi-Fi" and #define SECRET\_PASS = "Password of Wi-Fi"
